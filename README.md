# OCPP Charger Simulator

## To run the program:

  - ### Install the Node modules
    `npm install`
  
  - ### Run the programe
    `node OCPPEndDevice.js`

## Key Features Added:
### Interactive Menu System

- Command 1: Simulate vehicle plug-in (changes status to Preparing)
- Command 2: Simulate RFID card swipe (independent charging)
- Command 3: Stop current charging session
- Command 4: Change back to Available status
- Command 5: Show current status
- Command 6: Show detailed session info
